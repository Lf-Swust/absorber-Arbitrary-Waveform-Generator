# absorber-Arbitrary-Waveform-Generator
absorber-Arbitrary Waveform Generator

项目概要：过FIR低通滤波滤除超过设定的高频波，任意波形发生器通过接受和判断uart接口的信息数据生成对应的正弦波、三角波、方波，周期不能通过uart数据改变。

使用工具版本：Vivado 2018.3

板卡型号与外设列表：EGO1、ADC、DAC、示波器

仓库目录介绍：

             Sourcecode：存放源代码。
